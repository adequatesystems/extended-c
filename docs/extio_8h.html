Append to a character string buffer.

Calculates the remaining available space in buf using bufsz and strlen(buf), and appends at most the remaining available space, less 1 (for the null terminator).

Parameters
bufPointer to character string buffer
bufszSize of buf, in bytes
fmtPointer to null-terminated format string
...arguments specifying additional data to print per format
Returns
(int) Number of characters (including the null terminator) which would have been appended to buf if bufsz was ignored.
