◆ rand16fast()

word32 rand16fast ( void  )

Fast 16-bit PRNG using internal state seed Lseed.

Based on Dr. Marsaglia's Usenet post of a linear congruential generator.

Returns
Random number range [0, 65535].
Warning
Exhibits an increase in PRNG generation at the cost of statistical randomness, when compared to rand16().

Shuffle a list[count] of size byte elements.

Uses Durstenfeld's implementation of the Fisher-Yates shuffling algorithm.

Note
Shuffle count is bound to 16 bits due to rand16().
Set random seed with srand16() before use.

◆ shufflenz()

void shufflenz ( void *  list,
size_t  size,
size_t  count 
)

Shuffle a list[count] of non-zero, size byte elements.

A zero value marks the end of list, and shuffling does NOT occur for said value onwards.
Uses Durstenfeld's implementation of the Fisher-Yates shuffling algorithm.

Note
Shuffle count is bound to 16 bits due to rand16().
Set random seed with srand16() before use.
