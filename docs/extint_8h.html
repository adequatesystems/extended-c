Detailed Description

Extended integer support.

Emulates the C Standard library header file <stdint.h> and extends on it's functionality.

For license information, please refer to ../LICENSE.md
Note
Number Constants for INT#_MAX have been removed due to an inability to determine an adequate naming scheme that does not conflict with the C Standard Library headers, "stdint.h" and "inttypes.h". If Number Constants for for INT#_MAX values are required, use exact number literals, or define the following:
// Where '#' represents bit width; 8/16/32/64
#define INT#_MAX = ( WORD#_MAX / 2 )
#define INT#_MIN = ( ( WORD#_MAX / (-2) ) - 1)

For 32-bit machine compatibility, either avoid the use of 64-bit elements or provide alternative routines separated by #ifdef's testing for the HAS_64BIT definition...
void someFunction(void)
{
#ifdef HAS_64BIT
// ... 64-bit routine
#else
// ... 32-bit routine
#endif
}
