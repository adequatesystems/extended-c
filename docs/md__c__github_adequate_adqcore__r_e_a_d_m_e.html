Extended C/C++ Library

Originally being derived from the Mochimo Codebase, this repository contains core C/C++ support intended for use in ongoing projects at Adequate Systems LLC.

Installation

The Extended C/C++ Library was designed to be included in other projects as a Git Submodule.

cd project-repo
git submodule add https://github.com/adequatesystems/extc include/

For C projects using Adequate's C Project Template, it is recommended to add submodules to the include/ directory of your project for plug-and-play compatibility with existing Makefiles.
