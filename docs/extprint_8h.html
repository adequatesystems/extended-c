Print a clean message to stdout.

When printing to screen, it's possible to get garbled text due to movement of the cursor by psticky(), so this splits the resulting string into lines and clears right after every line.

Parameters
fmtA string format (or message) for printing
...Variable arguments supporting the format string
Note
Prints to screen, regardless of specified print level.
